options参数

常用参数调用示例


$("#id").AppleSlider({
	imgItem:'.garllery_item',
	autoplay : true, 
	imgNum : 3, 
	autoTime : 5000 
})
参数 默认值 说明
imgItem 'gallery_item' 轮播内容列表的class
autoplay true 是否为自动轮播
imgNum 3 轮播图数量，默认值为3，没有最大值限制
autoTime 5000 自动轮播时间间隔
_index 0 当前轮播图绝对序号 (不可更改)
slide_state 0 轮播状态值,默认为可执行轮播动画,为1时不能执行轮播动画,每次轮播动画执行完毕前不可连续执行动画 (不可更改)
sildeTimer null 自动轮播定时器 (不可更改)
stateTimer null 状态值改变定时器 (不可更改)
